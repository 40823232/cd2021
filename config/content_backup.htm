coppeliasim

如果要讓零件產生碰撞要開啟respondable，上排表示同個基準的零件碰撞，下排是任何基準的零件碰撞

要讓零件動就要開啟dynamic

如果要將物件隱藏要把camera visibility layers給取消

這是將組合圖拆解，方便對各零件操作

要將兩物體綁在一起時，要按apply to selection(旋轉和移動)

set GIT_SSH=%Disk%:\putty\plink.exe
